Este es un demo de el uso de typescript.
Typescript es un superset de javascript, lo que sinifica que en lugar de ser un nuevo lenguaje, se convierte en un javascript con los esteroides de la orientacion a objetos. Typescript se puede usar en cualquier tipo de proyecto donde se usen tecnologias web ya que cada que typescript se compila, genera archivos (o archivo) de javascript. que corresponden a los estandares actuales del ECMAScript 2015.
